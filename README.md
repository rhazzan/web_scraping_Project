# Web Scraping Book Downloader

## Overview
This project is a Python web scraping tool built to collect textbook download links from a website and organize them into a clean Excel spreadsheet.

The idea came from my experience trying to learn Machine Learning and Data Science through books. While searching for recommended textbooks online, I found a website that had many of the books I needed, but navigating through the ads was frustrating.

To simplify the process, I built a scraper that extracts the book information and stores everything in an organized format so users can quickly access the download links.

---

## Features
- Scrapes book information from multiple pages
- Extracts direct download links
- Handles pagination
- Cleans and structures scraped data
- Exports results into an Excel spreadsheet
- Reduces the need to manually navigate through ads

---

## Technologies Used
- Python
- Requests
- BeautifulSoup4
- Pandas
- OpenPyXL

---

## Challenges Faced
During development, I encountered several challenges including:

- Different page structures across the website
- Hidden or dynamically loaded download links
- Random request failures and timeouts
- Data cleaning and formatting issues
- Handling incomplete or missing fields

Solving these problems helped improve my understanding of web scraping, debugging, and data processing.

---

## What I Learned
This project helped me improve my skills in:

- Web scraping
- HTML parsing
- Handling HTTP requests
- Data cleaning
- Working with Excel files in Python
- Debugging large scraping workflows

---

## Output
The final output is an Excel spreadsheet containing:
- Book titles
- Authors
- Download links
- Other relevant metadata
